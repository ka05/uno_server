# WAD UNO
Multiplayer UNO Web App for Web Application Development Class at RIT

# Getting started

- Clone repo
- Install node and npm

```shell
npm install
```

Copy `.env.default` to `.env` and setup database connection env vars.

```shell
cp .env.default .env
```
